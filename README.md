# DAT | Desirability Assessment Tool

## A high-performance, premium web application designed to validate problem statements and case studies using the DAT framework.

## Pillars of Success
The DAT is built around four core pillars:
1. **Problem Understanding**: Assessing customer awareness and problem clarity.
2. **Solution Search**: Evaluating customer effort and frustration with current solutions.
3. **Willingness to Pay**: Determining value perception and price sensitivity.
4. **Market Viability**: Quantifying TAM, SAM, and SOM.

## Scoring Guide
- **Low Desirability (9-19)**: Significant issues to address. Re-evaluate relevance.
- **Moderate Desirability (20-32)**: Potential exists. Needs refinement in customer understanding.
- **High Desirability (33-45)**: Strong potential. Ready for solution development.

## Features
- **Real-time Scoring**: Instant visual feedback via a dynamic gauge.
- **Micro-animations**: Smooth transitions and number counters.
- **Premium Design**: Glassmorphism, neon accents, and responsive layout.
- **Export Ready**: Clean print styles for PDF reports.

## How to Use
1. Enter your **Problem Statement**.
2. Rate each of the 9 questions on a scale of **1 to 5**.
3. Add **Justification Notes** for each rating.
4. Review the **Overall Desirability Score**.
5. Click **Export Report** to save your assessment.
