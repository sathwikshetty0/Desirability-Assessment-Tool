## 📌 Overview

**DAT (Desirability Assessment Tool)** is a high-performance innovation validation platform designed to transform subjective startup ideas into structured, data-backed strategic insights.

It bridges the gap between **idea conception** and **capital deployment** by quantifying venture desirability through a 9-dimension scoring framework across four strategic pillars.

> DAT replaces intuition-driven validation with evidence-backed decision-making.

---

## 🎯 Why DAT?

Early-stage ventures fail not due to poor execution — but due to poor validation.

DAT helps teams:

* ✅ Quantify qualitative market signals
* ✅ Identify structural risks before building
* ✅ Replace “gut-feel” decisions with measurable logic
* ✅ Generate investor-ready strategic briefs instantly
* ✅ Centralize innovation data across teams

# 🧠 The DAT Framework

DAT evaluates ventures across **4 Strategic Pillars** and **9 Critical Dimensions**.

---

## 1️⃣ Problem Pillar — *Customer Awareness*

| Dimension | Key Question                                              |
| --------- | --------------------------------------------------------- |
| Awareness | Do customers recognize this problem?                      |
| Clarity   | Is the problem clearly understood from their perspective? |

🔍 Measures whether the market *acknowledges* the pain point.

---

## 2️⃣ Intent Pillar — *Customer Effort & Friction*

| Dimension     | Key Question                                 |
| ------------- | -------------------------------------------- |
| Search Effort | Are customers actively trying to solve it?   |
| Frustration   | How painful is the current solution process? |

📈 Behavioral proof of demand.

---

## 3️⃣ Value Pillar — *Economic Perception*

| Dimension         | Key Question                            |
| ----------------- | --------------------------------------- |
| Value Perception  | Is this a painkiller or a vitamin?      |
| Price Sensitivity | Are customers willing to pay a premium? |

💰 Validates monetization viability.

---

## 4️⃣ Viability Pillar — *Market Scale*

| Dimension | Key Question                  |
| --------- | ----------------------------- |
| TAM       | Total Addressable Market      |
| SAM       | Serviceable Available Market  |
| SOM       | Serviceable Obtainable Market |

📊 Ensures the opportunity is scalable and realistic.

# 📊 Scoring System

* 9 Dimensions
* 1–5 Rating Each
* Maximum Score: **45**

### 🎯 Strategic Banding (v3.0)

| Score | Interpretation        | Action                 |
| ----- | --------------------- | ---------------------- |
| 0–19  | 🚨 Critical Risk      | Urgent Strategic Pivot |
| 20–32 | ⚖️ Moderate Potential | Refine & Revalidate    |
| 33–45 | ✅ High Desirability   | Validated Opportunity  |

> Every rating requires **evidence-based justification** — ensuring intellectual rigor.

# ☁️ Enable Google Sheets Integration

## Step 1 — Create Google Sheet

* Create new Google Sheet
* Go to **Extensions → Apps Script**

## Step 2 — Add Backend Script

* Paste `google_sheets_script.js`
* Save as **“DAT Backend”**

## Step 3 — Deploy

* Deploy → New Deployment
* Type: Web App
* Execute as: Me
* Access: Anyone

Copy the Web App URL.

## Step 4 — Connect Frontend

In `script.js`:

```javascript
const SHEET_URL = "https://script.google.com/.../exec";
```

Paste your deployed URL.

Done. 🎉
